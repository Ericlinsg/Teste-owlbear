# Combat Helper – Owlbear Rodeo

Extensão para Owlbear Rodeo 2.0 que permite:

- Registrar HP e AC em tokens
- Realizar ataques via menu de contexto
- Comparar automaticamente com AC
- Aplicar dano diretamente no HP

## Como usar

1. Suba este repositório no GitHub
2. Ative GitHub Pages (branch main)
3. Copie a URL do `manifest.json`
4. No Owlbear:
   - Extensions
   - Add Extension
   - Cole a URL

## Uso em jogo

- Botão direito no token → Registrar Combate
- Botão direito no token → Atacar

Compatível com qualquer sistema baseado em d20.
